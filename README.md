# Store-Analytics-Dashboard

## Overview
This project analyzes retail store performance using sales, inventory, and return data. It generates KPIs, forecasts future sales using regression, and provides business recommendations for each store.

The goal is to simulate a real-world retail analytics workflow used in consulting and business intelligence roles.

---

## Key Features

- KPI calculations:
  - Average sales
  - Return rate
  - Inventory turnover

- Sales forecasting using Linear Regression
- Rule-based recommendation engine:
  - Increase stock
  - Investigate returns
  - Reduce inventory
  - Maintain strategy

- Interactive visualizations using Plotly
- Store-level performance breakdown

---

## Tech Stack

- Python
- Pandas, NumPy
- Scikit-learn
- Plotly

---

## How It Works

1. Sales history is used to compute trends
2. Linear regression predicts next-month sales
3. Business rules generate actionable recommendations
4. Visualization shows trends and forecast per store

---

## Example Output

Each store returns:
- KPI summary
- Forecasted sales
- Suggested business action
- Interactive sales trend chart
